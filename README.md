# Hyde Tech Solutions - Public Assets

Publicly hosted assets for Hyde Tech Solutions.

## Purpose

This repository contains public-facing assets like logos and branding materials that need to be accessible via HTTPS URLs for:

- BIMI email authentication
- Website embedding
- Third-party integrations

## Contents

- `logo/` - Logo files in various formats
- `icons/` - Icon files

## Usage

Files in this repo are accessible via:

```
https://raw.githubusercontent.com/shawnhyd3-crypto/hyde-tech-assets/main/[path-to-file]
```
